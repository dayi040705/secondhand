# 校园二手交易平台 (Secondhand Trading Platform)

基于 Spring Boot + MyBatis + MySQL 的校园二手交易平台，实现用户注册登录、商品发布（含图片上传）、分类筛选、关键词搜索、商品评论、校内认证、站内私信、订单管理（购买/确认收货）以及管理员后台管理等功能。

## 技术栈

- **后端**：Spring Boot 2.7.x, MyBatis, MySQL 8.0
- **前端**：Thymeleaf, Bootstrap 5 (Bootswatch Quartz 主题)
- **构建工具**：Maven
- **开发工具**：IntelliJ IDEA

## 环境要求

- JDK 17 或更高版本
- MySQL 8.0 或 5.7
- Maven 3.6+ (IDEA 内置亦可)
- Git（可选）
